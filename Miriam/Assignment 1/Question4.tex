This network is not sound because:
\begin{enumerate}
	\item \textbf{safeness}, places cannot hold multiple tokens at the same time.
	That is true\\
	\item \textbf{proper completion}, if the sink place is marked, all other places
	are empty. This is true following the definition.\\
	\item \textbf{option to complete}, it is always possible to reach the marking
	that marks just the sink place. This is not true, because a never can be
	reached, since therefore start and d would have to fire. So it is not
	possible to enable e and than also not to reach the sink place.\\
	\item \textbf{absence of dead parts}, for any transition
	there is a firing sequence enabling it. Same reason than above.
\end{enumerate}
